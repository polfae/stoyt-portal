# Stoyt Portal v5.5.0

Responsible person management and task-column alignment.

Updates:
- Admin and public users can rename responsible people from the task editor
- Admin and public users can delete responsible people from the whole competition from the task editor
- Renaming a responsible person updates task assignments, role assignments and filters
- Deleting a responsible person removes them from tasks, roles and filters
- Admin can also rename/delete responsible people in the competition edit modal
- Task cards now use fixed columns so title, responsible people, deadline and comments stay aligned
- Applies to both admin and public competition views
- Firestore saving, public password access, admin login, planner and calculator are kept

Open `index.html` in your browser.
